#

奇偶校验任务

这将从论文《循环神经网络的自适应计算时间》中为奇偶校验任务创建数据。

奇偶校验任务的输入是一个带有0's 和1's 的向量。输出是−1's 的1奇偶校验——如果有，则为 1是的奇数1，否则为零。输入是通过使矢量中的随机数量的元素为1或而生成−1的。
