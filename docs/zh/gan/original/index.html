#

生成对抗网络 (GAN)

这是生成对抗网络的实现。

生成@@

器G(zz;θg​)生成与数据分布相匹配的样本，而鉴别器则D(xx;θg​)给出来自数据而不是xx来自数据的概率G。

我们在具有值功能的双人最小最大游戏中G同时进行训练DV(G,D)。

Gmin​Dmax​V(D,G)=Exx∼pdata​(xx)​[logD(xx)]+Ezz∼pzz​(zz)​[log(1−D(G(zz))]

pdata​(xx)是数据的概率分布，而pzz​(zz)概率分zz布则设置为高斯噪声。

这个文件定义了损失函数。这是一个 MNIST 示例，其中包含两个用于生成器和鉴别器的多层感知器。
