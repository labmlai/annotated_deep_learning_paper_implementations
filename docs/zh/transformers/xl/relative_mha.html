Ajabs​​=linq​(Xiq​+Pi​)⊤link​(Xjk​+Pj​)=AQi⊤​Kj​​+BQi⊤​UjK​​+CUiQ​⊤Kj​​+DUiQ​⊤UjK​​​

其中Qi​,Kj​，是原始嵌入的线性变换Xiq​,Xjk​，UiQ​,UjK​是绝对位置编码的线性变换Pi​,Pj​。

他们认为，无论查询的位置如何，对给定键的关注都应该相同。因此，CUiQ​⊤Kj​​用常量替换Cv⊤Kj​​。

Rk​

108        key_pos_emb = self.key_pos_embeddings[self.P - key.shape[0]:self.P + query.shape[0]]

Sk​

(A+C)i,j​=Qi⊤​Kj​+v⊤Kj​

117        ac = torch.einsum('ibhd,jbhd->ijbh', query + query_pos_bias, key)
