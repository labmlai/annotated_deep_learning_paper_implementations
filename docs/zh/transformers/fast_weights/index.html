快速称重变压器

论文《线性变形金刚是秘密的快速加权记忆系统》发现了线性自注意力和快速加权系统之间的相似之处，并据此修改了自我注意力更新规则。它还引入了一个更简单但有效的内核函数。

作者提供了该论文的正式实现，包括他们在论文中比较的其他变体。

快速举重

考虑一个输入序列{x(i)}i=1L​或长度L，每个步骤都是大小向量din​；即x∈Rdin​快速权重模型在每个步骤生成权重矩阵以生成输出{y(i)}i=1L​，y∈Rdout​

a(i),b(i)W(i)y(i)​=Wa​x(i),Wb​x(i)=σ(W(i−1)+a(i)⊗b(i))=W(i)x(i)​

⊗是外积 (a⊗b=ab⊤)，其中两个向量的元素相互乘以得出矩阵。σ是一个激活函数。Wa​并且Wb​是可训练的权重（参数）。W(i)是在每一步生成的快速权重。

线性自我注意力

原始变压器的自我注意力是，（dk​1​为了清楚起见，省略了）

y(i)​=[v(1),v(2),...,v(i)]softmax([k(1),k(2),...,k(i)]⊤q(i))=j=1∑i​∑j′=1i​κ(k(j′),q(i))v(j)κ(k(j),q(i))​​

在哪里κ(k,q)=exp(k⋅q)

线性化自我注意力背后的想法是κ用不同的内核替换 softmax 内核，κ′这样我们就可以更快地计算出自我注意力函数的分母：

κ′(k,q)=ϕ(k)⊤ϕ(q)

这给了

y(i)​=(∑j′=1i​ϕ(k(j′)))ϕ(q(i))(∑j=1i​v(j)⊗ϕ(k(j)))ϕ(q(i))​​

使用W(i)=∑j=1i​v(j)⊗ϕ(k(j))和z(i)=∑j=1i​ϕ(k(j))，我们可以有效地计算它们：

W(i)z(i)y(i)​=W(i−1)+v(i)⊗ϕ(k(i))=z(i)+ϕ(k(i))=z(i)⋅ϕ(q(i))1​W(i)ϕ(q(i))​

这与快速称重非常相似。

确定性无参数项目 (DPFP)

这是本文中ϕ引入的新投影函数。DPFP 从维度dkey​到维度的投ddot​=2dkey​ν影，其中ν∈1,2,...,2dkey​−1是一个超参数。k

ϕ2dkey​(i−1)+j​(k)=ReLU([k,−k])j​ReLU([k,−k])i+j​

where[k,−k] 是k和的串联−k，用于给出大小为2dkey​i∈1,2,...,ν、和j∈1,2,...,2dkey​。xi​是 vector 的i第-th 个元素x，如果大i于 vector 中的元素数量，则会滚动x。

基本上，它通过乘以[k,−k]移位的元素来创建一个新的向量i。

这将生成稀疏投影（只有少数元素为非零）和正交投影（ϕ(k(i))⋅ϕ(k(j))≈0对于大多数，i,j除非k(i)和phik(j)非常相似。

快速举重注意

本文介绍了一种新的计算更新规则W(i)。模型首先检索与键vˉ(i)配对的当前值k(i)。然后存储检索到v(i)new​的值vˉ(i)和输入的组合v(i)。

k(i),v(i),q(i)vˉ(i)β(i)v(i)new​W(i)y(i)​=Wk​x(i),Wv​x(i),Wq​x(i)=W(i−1)ϕ′(k(i))=σ(Wβ​x(i))=β(i)v(i)+(1−β(i))vˉ(i)=W(i−1)+v(i)new​⊗ϕ′(k(i))=W(i−1)+β(i)(v(i)−vˉ(i))⊗ϕ′(k(i))=W(i)ϕ′(q(i))​

214        self.key = PrepareForMultiHeadAttention(d_model, heads, self.d_k, bias=False)
215        self.value = PrepareForMultiHeadAttention(d_model, heads, self.d_k, bias=False)
#

每个头的插值权重函数σ(Wβ​x(i))

231    def forward(self, x: torch.Tensor):
#

获取步数L

233        seq_len = x.shape[0]
#

ϕ′(q(i))适用于所有台阶和头部

235        query = self.phi(self.query(x))
#

ϕ′(k(i))适用于所有台阶和头部

237        key = self.phi(self.key(x))
#

v(i)适用于所有台阶和头部

239        value = self.value(x)
#

β(i)适用于所有台阶和头部

241        beta = self.interpolation_weight(x)
#

W(0)

244        weights = key.new_zeros((key.shape[1], key.shape[2], value.shape[3], key.shape[3]))
#

存储输出的列表y(i)

246        outputs = []
#

遍历各个步骤

249        for i in range(seq_len):
#

vˉ(i)=W(i−1)ϕ′(k(i))

251            value_existing = torch.einsum('bhvk,bhk->bhv', weights, key[i])
#

W(i)=W(i−1)+β(i)(v(i)−vˉ(i))⊗ϕ′(k(i))

256            weights = weights + torch.einsum('bhv,bhk->bhvk', beta[i] * (value[i] - value_existing), key[i])
#

259            y = torch.einsum('bhvk,bhk->bhv', weights, query[i])
#

合并多个头部并追加到outputs 

262            outputs.append(y.reshape(y.shape[0], -1))
#

265        x = torch.stack(outputs)
#
