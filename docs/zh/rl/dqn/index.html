目标网络 🎯

为了提高稳定性，我们使用经验回放，从以前的经验中随机抽样U(D)。我们还使用具有一组单独参数的 Q 网络θi−​来计算目标。θi−​定期更新。这是根据论文《通过深度强化学习进行人体水平控制》得出的。

所以损失函数是，Li​(θi​)=E(s,a,r,s′)∼U(D)​[(r+γa′max​Q(s′,a′;θi−​)−Q(s,a;θi​))2]

双Q重学习

上述计算中的最大值运算符使用相同的网络来选择最佳动作和评估值。也就是说，a′max​Q(s′,a′;θ)=Q(s′,argmaxa′​Q(s′,a′;θ);θ)我们使用双重Q-Largmax earningθi​，其中取自值，取自值θi−​。
