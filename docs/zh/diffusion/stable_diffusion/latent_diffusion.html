#

潜在扩散模型

潜在扩散模型使用自动编码器在图像空间和潜在空间之间进行映射。扩散模型适用于潜在空间，这使得训练变得容易得多。它基于带有潜在扩散模型的纸质高分辨率图像合成。

它们使用预训练的自动编码器，在预训练的自动编码器的潜在空间上训练扩散 U-Net。

有关更简单的扩散实现，请参阅我们的 DDPM 实现。我们对αt​βt​时间表等使用相同的符号。
