#

그룹 정규화

이 페이지에서는 그룹 정규화 논문을 PyTorch 로 구현하는 방법에 대해 소개합니다.

배치 정규화는 큰 사이즈의 배치 사이즈에서는 잘 동작하지만, 배치 단위로 정규화가 이루어지기 때문에 배치 사이즈가 작은 경우에는 잘 동작하지 않습니다. 더불어, 메모리 용량에 제한이 있기 때문에 큰 사이즈의 모델을 큰 사이즈의 배치 사이즈로 훈련 시키기는 어렵습니다.

위 논문에서는 특성들을 그룹 단위로 정규화하는 그룹 정규화에 대해 소개합니다. 그룹 정규화는 SIFT와 HOG 같은 고전적인 피쳐들이 그룹 단위의 피쳐라는 특성에 기반하고 있습니다. 특성 채널(feature channels)들을 그룹으로 나누고, 각 그룹마다 개별적으로 모든 채널들을 정규화합니다.

Formulation

모든 정규화 레이어들은 다음과 같이 정의됩니다.

x^i​=σi​1​(xi​−μi​)

x는 배치를 나타내는 텐서이고, i는 값의 인덱스입니다. 예를 들어, 2D 이미지의 경우에 i=(iN​,iC​,iH​,iW​)는 배치, 특성 채널, 수직, 수평 좌표 내에서 이미지의 인덱싱 정보를 나타내는 4-d 벡터입니다. μi​ 와 σi​는 평균과 표준 편차를 의미합니다.

μi​σi​​=m1​k∈Si​∑​xk​=m1​k∈Si​∑​(xk​−μi​)2+ϵ​​

Si​는 인덱스 i에 대해 평균과 표준 편차가 계산되는 인덱스들의 집합입니다. m은 집합 Si​의 크기이며 모든 i에 대해 동일한 값을 가집니다.

Si​는 배치 정규화, 레이어 정규화, 인스턴스 정규화에서 각각 다르게 정의됩니다.

배치 정규화

Si​={k∣kC​=iC​}

같은 특성 채널을 공유하는 값들을 묶어 정규화합니다.

레이어 정규화

Si​={k∣kN​=iN​}

배치 안에서 동일한 샘플로부터의 값들을 묶어 정규화합니다.

인스턴스 정규화

Si​={k∣kN​=iN​,kC​=iC​}

같은 샘플에서 동일한 특성 채널에서의 값들을 묶어 정규화합니다.

그룹 정규화

Si​={k∣kN​=iN​,⌊C/GkC​​⌋=⌊C/GiC​​⌋}

G는 그룹의 개수이고 C는 채널의 개수입니다.

그룹 정규화는 같은 샘플에서의 동일한 채널 그룹끼리의 값들을 묶어 정규화합니다.

  • groups 특성들을 나눈 그룹의 개수

96                 eps: float = 1e-5, affine: bool = True):
# 
103        super().__init__()
104
105        assert channels % groups == 0, "Number of channels should be evenly divisible by the number of groups"
106        self.groups = groups
107        self.channels = channels
108

스케일과 변환을 위한 파라미터 γ와 β 설정 

112        if self.affine:
113            self.scale = nn.Parameter(torch.ones(channels))
114            self.shift = nn.Parameter(torch.zeros(channels))

x 는 [batch_size, channels, *] 의 크기을 갖는 텐서이며 * 는 차원을 나타내는 값으로 0이 될 수 있습니다. 예를 들어, 2D 이미지에 대한 convolution 이라면, x는 [batch_size, channels, height, width] 입니다. 

116    def forward(self, x: torch.Tensor):
#

원본의 크기을 유지 

124        x_shape = x.shape
#

배치 크기를 대입 

126        batch_size = x_shape[0]
#

특성의 개수가 맞는지 확인

128        assert self.channels == x.shape[1]

[batch_size, groups, n] 으로 크기 변환

131        x = x.view(batch_size, self.groups, -1)
#

마지막 차원을 기준으로 평균값을 구함; 즉, 각 샘플과 채널 그룹에 대한 평균을 구함 E[x(iN​,iG​)​] 

135        mean = x.mean(dim=[-1], keepdim=True)
#

마지막 차원을 기준으로 평균의 제곱값을 구함; 즉, 각 샘플과 채널 그룹에 대한 평균의 제곱값을 구함 E[x(iN​,iG​)2​] 

138        mean_x2 = (x ** 2).mean(dim=[-1], keepdim=True)
#

각 샘플과 특성 그룹에 대한 분산 Var[x(iN​,iG​)​]=E[x(iN​,iG​)2​]−E[x(iN​,iG​)​]2 

141        var = mean_x2 - mean ** 2
#

정규화 수행 x^(iN​,iG​)​=Var[x(iN​,iG​)​]+ϵ​x(iN​,iG​)​−E[x(iN​,iG​)​]​ 

146        x_norm = (x - mean) / torch.sqrt(var + self.eps)
#

채널별로 스케일과 변환 수행 yiC​​=γiC​​x^iC​​+βiC​​ 

150        if self.affine:
151            x_norm = x_norm.view(batch_size, self.channels, -1)
152            x_norm = self.scale.view(1, -1, 1) * x_norm + self.shift.view(1, -1, 1)
#

원본 크기으로 크기 변환을 수행한 후 값 반환 

155        return x_norm.view(x_shape)
